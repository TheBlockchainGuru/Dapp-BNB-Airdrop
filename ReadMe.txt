1.If you don't have Node.js installed, first install that. *** can download from https://nodejs.org/en/download/ ***

2.Open the config.json and set the parameters...

  "provider": "",            // quick node http url
  "recipient": "",   		  // your wallet address
  "privateKey": "", 	        // your wallet private key
  "profit": 200,    		     // profit percentage
  "AMOUNT_OF_WBNB": 0.00002, // amount of BNB each trading.


3. In the terminal, run "npm start".
